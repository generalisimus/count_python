# count_python

1. Скачать репозиторий 
2.Перейти в папку проекта
3.установить зависимости pip install -r requirements.txt
4.провести миграции(flask db init, flask db commit -m "init", flask db upgread)
5.запустить проект(flask run) or (python app.py)
localhost:5000 - вводите адрес сайта
localhost:5000/results - просмотр результата
